Free Multipurpose WooCommerce WordPress Theme

== Description ==

Description: Envo Multipurpose is a fast, clean and modern-looking responsive multipurpose WooCommerce theme for WordPress. The theme is fully widgetized, so users can manage the content by using easy to use widgets. The theme works with any drag & drop page builder like Elementor, KingComposer, Beaver Builder, SiteOrigin, Divi, Visual Composer, etc. Theme is compatible with all major plugins like WooCommerce, Yoast, Contact Form 7 etc. You can build any type of websites like shop, fashion store, portfolio, business agencies, corporate, education, university portal, consulting, restaurant, medical and so on. Envo Multipurpose is SEO friendly, WPML, translation and RTL ready. Demos and demo import: https://envothemes.com/envo-multipurpose/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= Theme Features Usage =
All available options can be used from Appearance->Customize
To setup homepage go to Customizer->Widgets (or Appearance->Widgets) and add EM:.... widgets into "Homepage Widgetized Area"
Theme documentation: https://envothemes.com/envo-multipurpose/documentation/
